# Product Overview

Ali Sinaee's animated portfolio website built with Flutter Web and deployed on Firebase.

## Core Features

- **Interactive animated menu** with smooth transitions and hover effects
- **Moving text backgrounds** with horizontal scrolling animations
- **Liquid glass visual effects** using custom GLSL shaders
- **11 custom fonts** for unique typography
- **Responsive design** across desktop, tablet, and mobile

## Performance Focus

This is a **performance-critical** application optimized for 60 FPS animations:
- WASM compilation for 2-3x faster execution
- Smart animation pausing to reduce rebuilds by 83%
- Memory management to prevent leaks
- Frame-aligned timing for smooth transitions

## Target Platform

Primary: **Web (Chrome/Edge)** with WebAssembly support
Secondary: Mobile (iOS/Android), Desktop (macOS/Windows/Linux)
